# Prodigy Modpack

## Overview
Prodigy is a lightweight and highly optimized Minecraft modpack designed as a solid foundation for modders, server admins, and casual players alike. Whether you're building your own custom modpack from scratch or just need a performant base for everyday gameplay, Prodigy delivers smooth performance with minimal overhead. Built on the latest Minecraft versions (compatible with 1.20+), it focuses on essential optimizations, quality-of-life improvements, and core utilities without bloating your game with unnecessary features.

## Key Features

- **Cross-Platform Compatibility**: Works seamlessly on Fabric or Quilt loaders, with optional Forge support via compatibility layers.
